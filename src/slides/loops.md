## Aren't "loops" the same thing?

[Ralph](https://ghuntley.com/ralph/) (Huntley) plus the 2026 "loop engineering" wave ([Cherny](https://www.youtube.com/watch?v=Sg7DdOfQCcY), [Steinberger](https://x.com/steipete/status/2063697162748260627)). I had Claude check whether the new loops are more RLM-like.

**No.** Same gate profile — the loop lives *outside* the agent. Loops scale an agent over *time*; RLMs scale one inference over *context*.
